How It Works
01
Client Sends Request
Browser sends an HTTP request to Nginx on port 80.
02
Nginx Decides
Nginx checks which backend server is next in the Round Robin sequence.
03
Request Forwarded
Nginx forwards the request to the selected backend server via proxy_pass.
04
Response Returned
The backend server responds and Nginx sends it back to the client.
05
Next Request Rotates
Next request goes to the next server. S1 → S2 → S3 → S1 → repeat.
06
Auto Failover
If a server dies, Nginx automatically skips it and uses the others.
